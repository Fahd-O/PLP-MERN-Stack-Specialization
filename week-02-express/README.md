
# Week 02 - Express

## Overview
This directory contains hands-on exercises and projects for learning Express.js, a minimal and flexible Node.js web application framework.

## Contents
- **Exercises**: Practical tasks covering Express fundamentals
- **Projects**: Mini-projects to apply learned concepts

## Topics Covered
- Setting up an Express server
- Routing and middleware
- Request/response handling
- Error handling
- Template engines
- Static file serving

## Getting Started

### Prerequisites
- Node.js installed
- Basic JavaScript knowledge

### Installation
```bash
npm install
```

### Running Examples
```bash
node app.js
```
